Häufig gestellte Fragen (FAQ) - WikiGUI

Willkommen im FAQ-Bereich für WikiGUI! Dieses Tool hilft Ihnen, erweiterte Suchanfragen für Wikipedia zu erstellen. Wenn Sie hier keine Antwort finden, beziehen Sie sich bitte auf die offizielle Wikipedia-Suchhilfe oder kontaktieren Sie den Support, falls verfügbar.

1. Erste Schritte

  • F: Was ist WikiGUI?
    A: WikiGUI ist ein webbasiertes Werkzeug, das Benutzer dabei unterstützt, komplexe Suchanfragen für Wikipedia zu erstellen, indem es eine intuitive Oberfläche für verschiedene Suchparameter bietet.
  • F: Ist die Nutzung von WikiGUI kostenlos?
    A: WikiGUI wird unter einer offenen Lizenz zur Verfügung gestellt (siehe Lizenzdatei). Die grundlegende Nutzung ist kostenlos. Bitte beachten Sie die Lizenz- und Nicht-Kommerziell-Nutzungsdateien für Details zu Nutzungsbeschränkungen und kommerzieller Nutzung.
  • F: Muss ich etwas installieren, um WikiGUI zu nutzen?
    A: Nein, WikiGUI ist eine Webanwendung. Sie können es direkt in Ihrem Webbrowser ohne Installation nutzen, vorausgesetzt, Sie haben eine Internetverbindung.
  • F: Wo finde ich die offizielle Wikipedia-Suchanleitung?
    A: Sie finden die offizielle Anleitung, indem Sie auf den Link in der Kopfzeile der Seite klicken, der Sie zur Suchhilfe-Dokumentation für die ausgewählte Wikipedia-Sprache weiterleitet.

2. Features & Funktionalität

  • F: Wie funktioniert das Feld "Alle diese Wörter"?
    A: Die Eingabe von Begriffen in das Feld "Alle diese Wörter" bedeutet, dass alle von ihnen in den Suchergebnissen vorhanden sein müssen (entspricht einem impliziten UND).
  • F: Was ist der Unterschied zwischen "Alle diese Wörter" und "Exakte Phrase"?
    A: "Alle diese Wörter" erfordert, dass jedes Wort vorhanden ist, aber nicht unbedingt zusammen. "Exakte Phrase" erfordert, dass die Wortfolge zusammen und in Anführungszeichen erscheint.
  • F: Wie kann ich bestimmte Wörter von meiner Suche ausschließen?
    A: Verwenden Sie das Feld "Ohne diese Wörter". Geben Sie die Wörter ein, die Sie ausschließen möchten, und Seiten, die eines dieser Wörter enthalten, werden nicht zurückgegeben.
  • F: Wie suche ich in einer bestimmten Kategorie?
    A: Sie können das Feld "In Kategorie" verwenden, um eine oder mehrere Kategorien anzugeben. Für Unterkategorien verwenden Sie das Feld "Tiefe Kategorie".
  • F: Was sind Namensräume?
    A: Namensräume sind verschiedene Bereiche innerhalb von Wikipedia (z.B. Artikel, Dateien, Kategorien, Hilfeseiten). Sie können bestimmte Namensräume auswählen, um Ihre Suchergebnisse einzugrenzen.
  • F: Wie funktioniert das neue Kategorie-Dropdown?
    A: Das "Kategorie"-Dropdown ermöglicht es Ihnen, eine vordefinierte Wikipedia-Kategorie (z.B. Wissenschaft, Geschichte) auszuwählen, um den Suchparameter "In Kategorie" schnell auszufüllen. Dies hilft, schnell auf breite Themenbereiche abzuzielen.

3. Fehlerbehebung & Support

  • F: Ich habe meine Suchparameter eingegeben, aber die "Generierte Suchzeichenkette" ist leer.
    A: Stellen Sie sicher, dass Sie mindestens einen Suchbegriff in den Hauptfeldern wie "Alle diese Wörter" oder "Exakte Phrase" eingegeben haben.
  • F: Die Suchergebnisse scheinen unvollständig oder falsch zu sein.
    A: Überprüfen Sie Ihre eingegebenen Parameter. Die Suchsyntax von Wikipedia kann komplex sein. Beziehen Sie sich auf den offiziellen Wikipedia-Suchhilfe-Link für detaillierte Informationen zur Funktionsweise der Parameter. Die Verwendung spezifischer Parameterkombinationen (wie `intitle:` vs. `intext:`) kann die Ergebnisse erheblich beeinflussen.
  • F: Wie melde ich einen Fehler oder schlage eine neue Funktion für WikiGUI vor?
    A: Bitte beziehen Sie sich auf das Repository des Projekts (falls verfügbar) für die Problemverfolgung oder kontaktieren Sie den Entwickler über [Kontaktmethode erwähnen, falls verfügbar, z.B. GitHub-Issues-Seite].
