Search Parameter Explanations

Main Query

Keywords: Enter your primary search terms here. By default, terms are combined with 'AND'. Use 'OR' for alternatives, or '-' before a word to exclude it. Use quotes for exact phrases.

Example: "Albert Einstein" OR "Isaac Newton" -biography

Exact Phrase: Search for a specific sequence of words.

Wikipedia Syntax: Encloses the text in double quotes (").

Example: "artificial intelligence"

Exclude Words: Remove results that contain certain words.

Wikipedia Syntax: Prefix the word with a hyphen (-).

Example: computer -laptop

Any Words: Find articles that contain at least one of the specified words.

Wikipedia Syntax: Use the 'OR' operator between words.

Example: apple OR orange

Wildcard: Use placeholder characters for unknown or variable parts of a word.

Wikipedia Syntax: Typically uses * (zero or more characters) or ? (a single character).

Example: comput*

Fuzzy Search: Find results even with slight misspellings or variations.

Wikipedia Syntax: Often uses ~ at the end of a word.

Example: color~

Scope & Location

In Title (intitle:): Restricts results to articles whose titles contain specific words.

Example: intitle:"United States"

Prefix (prefix:): Finds pages with a specific prefix in their title (e.g., in a specific namespace).

Example: prefix:User:

In Category (incategory:): Limits results to articles that are members of a specific category.

Example: incategory:"Artificial intelligence"

Deep Category (deepcat:): Includes articles in subcategories of the specified category.

Example: deepcat:"Science fiction"

Link From (linkfrom:): Finds articles that link to a specified page.

Example: linkfrom:Wikipedia:Village pump

Namespaces: Select specific Wikipedia namespaces (e.g., Main Article, User, File, Category) to search within.

Content & Technical

In Source (insource:): Searches within the wikitext content of pages, not just the displayed text. Useful for finding specific markup or code snippets.

Example: insource:/Â£{{Infobox/

Has Template (hastemplate:): Finds pages that use a specific template.

Example: hastemplate:"Infobox country"

File Type: Filter search results for specific file extensions (e.g., PNG, JPG, PDF).

Wikipedia Syntax: filetype:extension

Example: filetype:pdf

File Size: Specify minimum and maximum file sizes for image or media searches.

Wikipedia Syntax: imagesize:value or imagesize:min-max

Example: imagesize:5MP

Date Filters

Date After: Limits results to articles created or modified on or after a specific date.

Format: YYYY-MM-DD

Example: createdafter:2023-01-01 or touchedafter:2023-01-01

Date Before: Limits results to articles created or modified on or before a specific date.

Format: YYYY-MM-DD

Example: createdbefore:2024-12-31 or touchedbefore:2024-12-31
